# Копирование объектов и дашбордов

Скрипт выгружает все объекты и дашборды с указанных инстансов. Добавление инстанса для бекапа осуществляется путём его добавления в ```instance.json```. В процессе работы в директории скрипта будет создана папка с текущей датой и названием инстанса из ключа ```instance``` файла ```instance.json```. 

!Важно: Скрипт отправляет GET и POST запросы в **синхронном** режиме, обработка 4000 запросов займет около 70 минут.

# Требуемые библиотеки для работы скрипта:
requests - ``pip install requests``
